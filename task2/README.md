


# Arduino Simulation (Medium) 🎯


### Project Description
This project demonstrates controlling a **servo motor using a potentiometer** in Tinkercad.

A safety constraint is implemented to ensure that the servo motor **does not rotate beyond 68°**.  
If the potentiometer tries to exceed this limit:
- The servo is restricted to 68°
- An LED turns ON as a warning indicator

## Technical Details
### Technologies/Components Used
For Software:
- Arduino Sketch (C/C++)
- Tinkercad Circuits (Simulation Platform)

For Hardware:
- Arduino Uno  
- Servo Motor 
- Potentiometer (10kΩ)  
- LED  
- Resistor (220Ω)  
- Jumper wires
